Express Calculator!

This is the landing page to a calculator run using dynamic paths. This project is written using Node.js and Express. A "how-to" documentation is below.

Path Name:

All paths using this calculator need to have the form /operation/number/number . Operator should be the operation that you wish to perform. Currently this calculator supports the following operations:

  • Addition (Add)
  • Subtraction (subtact)
  • Multiplication (multiply)
  • Division (divide)
  • Exponentiation (pow)

The numbers that follow are the two numbers that you wish the operation to be performed on. NOTE: the order does matter.

Example

/subtact/13/2 would run 13-2
