Express Calculator!

This is the landing page to a calculator run using dynamic paths. This project is written using Node.js and Express. A "how-to" documentation is below.

Path Name:

All paths using this calculator need to have the form /operation/number/number . Operator should be the operation that you wish to perform. Currently this calculator supports the following operations:

  • Addition (Add)
  • Subtraction (subtact)
  • Multiplication (multiply)
  • Division (divide)
  • Exponentiation (pow)

The numbers that follow are the two numbers that you wish the operation to be performed on. NOTE: the order does matter.

Example

/subtact/13/2 would run 13-2

A specific write up for each operation is found below

Addition

Acceptable path name: /add/num1/num2

The output will be num1+num2

Example #1

/add/13/18 would return 31 (13+18)

Example #2

/add/-9/17 would return 8 (-9+17)

Subtraction

Acceptable path name: /subtract/num1/num2

The output will be num1-num2. Note that the order here will matter!

Example #1

/subtract/116/20 would return 96 (116-20)

Example #2

/subtract/89/100 would return -11 (89 - 100)

Example #3

/subtract/89/-100 would return 189 (89 - -100)

Multiplication

Acceptable path name: /multiply/num1/num2

The output will be num1*num2

Example #1

/multiply/13/18 would return 234 (13*18)

Example #2

/multiply/-9/17 would return -153 (-9*17)

Division

Acceptable path name: /divide/num1/num2

The output will be num1/num2

Example #1

/divide/24/2 would return 12 (24/2)

Example #2

/divide/-9/17 would return -0.529 (-9/17)

Exponentiation

Acceptable path name: /pow/num1/num2

The output will be num1num2

Example #1

/pow/8/2 would return 64 (1318)

Example #2

/pow/12/-3 would return 0.00057 (12-3 [the same as 1/123])
